Course Description

This course covers the design and analysis of efficient algorithms. Topics include sorting, searching, dynamic programming, greedy algorithms, and graph algorithms. Emphasis is placed on understanding the mathematical foundations and performance characteristics of algorithms.

Learning Outcomes

  • check_circle Analyze the time and space complexity of algorithms.
  • check_circle Implement classic sorting and searching algorithms.
  • check_circle Apply dynamic programming to solve complex optimization problems.

Technical Requirements

Proficiency in C++ or Python is required for laboratory assignments. Students should have access to a UNIX-like terminal environment.
