Course Description

This course covers algorithm design paradigms and asymptotic analysis, with practical implementation patterns for sorting, searching, dynamic programming, greedy methods, and graph algorithms. Students will learn to reason about correctness and efficiency.

emoji_objects

Learning Outcomes

  • Analyze time and space complexity using Big-O notation.
  • Design solutions using dynamic programming and greedy strategies.
  • Implement graph traversal, MST, and shortest-path algorithms.
  • Apply divide-and-conquer to classic algorithmic problems.
